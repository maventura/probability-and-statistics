



\begin{section}{Ejercicio 5}
Implemente las funciones simulacion mv(b, n), simulacion mom(b, n) y simulacion med(b, n)
que devuelve la aproximaci√≥n del ECM de cada uno de los estimadores correspondientes al b
y al n.

\begin{subsection}{Implementacion}~\\



\begin{verbatim}
simulacion_mom<-function(b,n)
{

nrep = 1000;
Bmoms = double();
X = double();

for (i in 1:nrep){
 X = runif(n, 0.0, b);
 Bmoms[i] <- Bmom(X);
}

momMean = mean(Bmoms);
momErr = abs(b-momMean);
momVar = var(Bmoms)
ECMBmom = momVar + (momErr^2);
return(ECMBmom);
}
\end{verbatim}
